# ☁️ Облачное хранилище (Cloud Storage Service)

REST-сервис для управления файлами в облачном хранилище с авторизацией по токенам. Сервис интегрируется с готовым frontend-приложением и предоставляет API для загрузки, скачивания, удаления файлов и просмотра списка файлов пользователя.

## ✨ Особенности

- 🔐 **Безопасная авторизация** - токен-базированная аутентификация с временными токенами
- 📁 **Управление файлами** - загрузка, скачивание, удаление и список файлов
- 🗄️ **Постоянное хранение** - PostgreSQL для метаданных, файловая система для хранения файлов
- 🐳 **Docker готовность** - полная контейнеризация с docker-compose
- 📊 **Масштабируемость** - поддержка больших файлов (до 50MB)
- 🔍 **Логирование** - подробное логирование всех операций
- ✅ **Тестирование** - unit-тесты с Mockito
